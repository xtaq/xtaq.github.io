PHP类型简介

By xiang tong

2015-09-09 更新日期:2015-09-09

文章目錄

PHP 支持 8 种原始数据类型。

四种标量类型：

  • boolean （布尔型）
  • integer （整型）
  • float （浮点型，也称作 double)
  • string （字符串）

两种复合类型：

  • array （数组）
  • object （对象）

最后是两种特殊类型：

  • resource （资源）
  • NULL （无类型）

可能还会读到一些关于“双精度（double）”类型的参考。实际上 double 和 float 是相同的，由于一些历史的原因，这两个名称同时存在。

变量的类型通常不是由程序员设定的，确切地说，是由 PHP 根据该变量使用的上下文在运行时决定的。

Note: 如果想查看某个表达式的值和类型，用 var_dump() 函数。
